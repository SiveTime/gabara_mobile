## ðŸŽ¯ PROJECT SCOPE

### Meetings Feature Scope

**Objective:** Enable mentors to create, manage, and track class meetings with attendance

**Included:**
- Create meetings with date, time, duration, type (online/offline)
- Edit and delete meetings
- Track attendance (present/absent/late)
- View attendance reports
- Meeting status management (scheduled/ongoing/completed/cancelled)
- Link meetings to classes

**Not Included (For Future):**

- Automatic attendance via QR code
- Meeting recordings
- Meeting reminders/notifications

### Assignments Feature Scope

**Objective:** Enable mentors to create assignments and track student submissions with grading

**Included:**
- Create assignments with deadline and max score
- Edit and delete assignments
- Student submission tracking
- Grading system with feedback
- Late submission detection
- Grade syncing to grades table
- Link assignments to meetings (optional)

**Not Included (For Future):**
- Plagiarism detection
- Peer review system
- Rubric-based grading
- Bulk grading operations
- Assignment templates
